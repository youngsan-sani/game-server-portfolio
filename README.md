# 게임 서버 개발 포트폴리오

PC 온라인 게임 서버 개발 경력 (2006.03 ~ ).  
게임 서버 컨텐츠 설계 및 개발, 라이브 서비스 운영 전반을 담당해 왔습니다.

## 기술 스택

- **언어**: C++
- **데이터베이스**: MySQL, Redis, ElasticSearch
- **메시징**: RabbitMQ

### 던전앤파이터

- 경매장
